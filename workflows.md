# Good Finances – Example Workflows

## Workflow 1 – First Budget Setup

1. User opens Good Finances and selects **“Set up my monthly budget”**.
2. AI assistant asks for net income, fixed costs, and typical variable spending.
3. Tool generates:
   - A simple budget view.
   - Suggestions on where to trim or rebalance.
4. User receives:
   - A saved template.
   - Optional printable or PDF version.

## Workflow 2 – Savings Goal

1. User selects **“Plan a savings goal”**.
2. Enters target amount and time frame.
3. Tool calculates monthly / weekly contributions.
4. AI suggests:
   - Trade-offs (e.g. small lifestyle cuts).
   - A simple tracking method with reminders.

## Workflow 3 – Year View

1. User opens **“Money calendar”**.
2. Adds big events (trips, renewals, tax bills).
3. Tool highlights:
   - High-spend months.
   - Recommended times to save or cut back.
4. AI assistant summarises key risks and suggestions.

These workflows guide how templates, AI interactions, and content should be designed.
